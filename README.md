# Spiky Dasher Game - Mobile PWA

A mobile-optimized version of the Spiky Dasher platformer game with PWA support.

## Features
- Mobile-friendly controls (virtual buttons)
- Progressive Web App (PWA) support
- Responsive design
- Installable on mobile devices
- Offline support via service worker
